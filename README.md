# 🌟 Sky キャンドル計算機（Sky Candle Calculator）

**Sky 星を紡ぐ子どもたち（Sky: Children of the Light）** のキャンドル獲得量を  
シンプルかつ直感的に管理できる Web 版計算ツールです。  
インストール不要・ブラウザだけで動作し、スマホにも完全対応しています。

---

## 🌐 公開ページ（GitHub Pages）

▶ **https://qk777.github.io/Sky_candle_calculator/**  

---

## ✨ 機能概要

### ✔ カテゴリ別チェック式で集計
- 各エリアのスポットを ON/OFF するだけで合計ライト数を自動計算  
- カテゴリごとの小計もリアルタイムで更新

### ✔ スマホ完全対応 UI  
- 小さな画面でも操作しやすい独自レイアウトを採用  

### ✔ 日曜日モード（値変更対応）  
- トグルスイッチで通常値／日曜日値を手動切り替え

### ✔ カスタム項目機能  
- スポット名やライト数を自由に編集可能  
- カテゴリ内に新規スポットを追加可能  
- ドラッグ＆ドロップで並び替え可能  
- Undo ボタンで直前の編集操作を 1 回だけ戻せる  
- 初期化ボタンでカテゴリを作者のデフォルト設定に戻せる  

### ✔ データは LocalStorage に保存  
- ブラウザ内に保存されるためプライバシー安全  
- サーバーへ送信されることは一切ありません


### ✔ 初期状態の説明
- 初期状態は作者の回収ルートに合わせた数値になっている
- 捨てられた地の値を編集した場合、日曜日モードの値も個別に設定が必要  
- パン屋1とパン屋2は合計で1000を上限としている  
  
## 🖥 使用技術

- **HTML / CSS / JavaScript（Vanilla）**
- **LocalStorage（データ永続化）**
- **Responsive Web Design**
- **GitHub Pages（ホスティング）**

フロントエンドのみで完結しているため、サーバー構築は不要です。
